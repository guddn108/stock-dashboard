# 주식/뉴스 대시보드 프로젝트

## 목표
매일 한국/미국 상위 5개 주요 뉴스 요약+분석 및 투자 종목 현황을 한눈에 보는 개인 대시보드.
- 모바일 친화적 웹페이지
- 캐나다 거주 기준, 한국 장 마감 시간에 자동 업데이트
- 서버 비용 없이 GitHub만으로 해결

---

## 확정된 기술 스택

| 역할 | 도구 | 비용 |
|------|------|------|
| 웹페이지 호스팅 | GitHub Pages | 무료 |
| 매일 자동 실행 | GitHub Actions | 무료 |
| 뉴스 수집 | NewsAPI | 무료 |
| 주가 데이터 | yfinance (Yahoo Finance) | 무료 |
| AI 분석 | Claude API (Anthropic) | 월 $1~2 |

---

## 동작 흐름

```
GitHub Actions 스케줄 트리거 (한국 장 마감 기준)
    ↓
Python 스크립트 실행
    ↓
뉴스 수집 (한국 + 미국) + 주가/재무 데이터 수집
    ↓
Claude API로 분석 (뉴스 요약+예측, 종목 현황+매수매도 구간)
    ↓
HTML 파일 생성 → GitHub Pages 배포
    ↓
사용자가 URL 접속 (폰/노트북 어디서나)
```

---

## 보안 처리
- API 키, 종목 티커 → GitHub Secrets에 저장 (코드에 노출 안됨)
- repo는 public이지만 실제 투자 종목은 숨겨짐
